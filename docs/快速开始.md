# 快速开始指南

## 📦 已安装完成

✅ 项目已配置完成,虚拟环境和依赖已安装

## 🚀 在 Claude Code 中使用

### 第一步:确认配置文件

已为你创建了 `CLAUDE.md` 配置文件,内容如下:

```markdown
# MCP Servers

## wechat-search
command: /Users/rick/Documents/AI产品开发/微信文章搜索摘要MCP/.venv/bin/python
args: ["/Users/rick/Documents/AI产品开发/微信文章搜索摘要MCP/mcp_server.py"]
```

### 第二步:启动 Claude Code

在任意目录下启动 Claude Code:

```bash
claude
```

Claude Code 会自动加载当前目录或全局的 MCP 配置。

### 第三步:使用搜索功能

直接用自然语言提问:

```
> 帮我搜索关于人工智能的微信文章

> 搜索"大模型应用"相关的公众号文章,获取5篇

> 查找最近关于Claude MCP的文章
```

Claude Code 会:
1. 自动识别需要调用搜索工具
2. 执行搜索并获取结果
3. 分析文章标题和摘要
4. 提供结构化总结

### 第四步:多轮对话

可以基于搜索结果继续提问:

```
> 这些文章主要讨论哪些观点?

> 第二篇文章的公众号是什么?

> 帮我总结这些文章的共同主题
```

## ⚙️ 配置说明

### 调整文章数量

编辑 `config.py`:

```python
DEFAULT_ARTICLE_COUNT = 3  # 默认获取数量
MAX_ARTICLE_COUNT = 10     # 最大数量限制
```

### 调整内容长度

```python
MAX_ARTICLE_LENGTH = 10000  # 单篇文章最大字符数
```

## 🔧 故障排查

### MCP 工具未显示

1. 检查 `CLAUDE.md` 文件路径是否正确
2. 确认使用绝对路径
3. 重启 Claude Code

### 搜索无结果

- 尝试更换关键词
- 检查网络连接
- 稍后重试(可能遇到反爬虫)
